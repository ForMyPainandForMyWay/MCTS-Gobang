# MCTS-Gobang

本项目是一个基于蒙特卡洛树搜索的五子棋桌面应用程序，开发之初的目的是为了机器学习与数据结构与基于国产系统的应用程序开发(Kylin & Qt)的课程设计。

在本项目中，使用了二级并行化加速搜索，首先对于每一个子节点开启一个独立的CPU线程，在此基础上每一个CPU线程GPU上开启大量子线程进行随机落子。GPU的并行计算部分使用了CUDA进行开发。

除此之外，本项目还使用Qt开发了图形化界面。
